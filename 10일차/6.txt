* 연결 리스트 자료 구조를 
  외부 클래스와 내부 클래스 관계를 설정해서 구현하기

1. 노드(Node) : 
    하나의 데이터만을 보관할 수 있는 변수
    예) 배열(x)
    다음에 연결될 데이터의 주소를 보관할 수 있는 참조 변수

class 외부클래스이름 {
   static class 내부클래스이름 {
       // 하나의 정수 데이터 만을 보관할 수 있는 일반 변수 선언
       int data;
       // 주소를 보관할 수 있는 참조 변수
       // -> class 내부클래스이름 : 자기 자신 : 자신의 모양을
       // -> 갖는 클래스의 주소를 보관할 수 있는 변수 :
       // -> self reference -> 자기 참조 변수
       내부클래스이름 참조변수이름;
   }
}
