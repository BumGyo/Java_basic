* 내부 클래스(Inner Class)란?
또는 중첩 클래스(Nested Class)
-> 클래스 간의 관계를 설정
-> 특정 클래스에서만 사용할 수 있는 전용 클래스

1) 기본 구조 : 
   class 새로운_클래스이름 {
       // 내부 클래스를 만듭니다!!
       class 내부_클래스이름 {
          // 다른 클래스에서 사용할 수 있는 클래스
       }
   }
-> 객체를 생성하는 순서 : 먼저 외부 클래스 이름을 사용해서
   메모리에 객체를 만들어야 합니다!!
   예) 새로운_클래스이름  참조변수이름 = new 새로운_클래스이름();

-> 내부 클래스는 메모리에 생성된 참조변수이름 이름을 사용해서
   메모리에 내부 클래스를 만들어야 합니다!!
   예) 새로운_클래스이름.내부_클래스이름 참조변수이름2;
       참조변수이름2 = 참조변수이름.new 내부_클래스이름();
   예) 내부_클래스이름 (직접은 불가능)
