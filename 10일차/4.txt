* 총 4가지로 외부 클래스와 내부 클래스 관계를 설정 가능

2. 클래스 변수 처럼 내부 클래스를 정의 가능
   -> 클래스 변수 : static 제한자 + 자료형 + 변수이름;

1) 구조:
   class 외부클래스이름 {
      static int s_a = 10;
      int a = 20;

      // static 제한자를 사용해서 내부 클래스를 정의하기
      static class 내부클래스이름 {
         // 외부 클래스에서 선언한 static 멤버 변수 또는
         // 메서드들만 사용 가능한 클래스 : s_a 변수는 사용 가능
         // 인스턴스 변수 a는 바로 사용 불가능
      }
   }
