* 자료 구조(Data Structure)

1. 스택(Stack) : 
    선입 후출 방식으로 데이터를 관리하는
    자료 구조
    -> 입력 순서의 반대로 메모리에 저장되어
       있는 데이터를 꺼내올 수 있는 자료 구조

2. 큐(Queue) : 선입 선출 방식으로 데이터를
   관리하는 자료 구조
   입력 순서대로 메모리에 저장되어 있는
   데이터를 꺼내올 수 있는 자료 구조
예) 일상 생활에서 줄 서기(버스, 전철 탈 때)
    물건을 쇼핑하신 후에 계산대에 있는 경우
    컴퓨터에서는 사용자 이벤트 처리(Event
    Handling)
    를 위해서 사용

1. 데이터를 보관할 수 있는 배열이 필요합니다.
   1) 정적 배열 : 
       자료형 배열이름 [ ] = 
          new 자료형[데이터 최대 갯수];
       예) int iarray[] = new int [100];
   2) 동적 배열 :
        java.util 패키지에 있는 ArrayList
        클래스를 사용한 배열
        구조) import java.util.ArrayList;
ArrayList<메모리에 저장할 데이터 타입>
   동적배열_참조변수이름 = 
 new ArrayList<메모리에 저장할 데이터 타입>();
