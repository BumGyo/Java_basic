* 데이터베이스를 사용하는 방법
  (새로 만들어서 사용하는 방법)

1. 테이블 만들기

2. 테이블에 데이터를 저장하기
-> insert sql 명령어를 사용하기
1) insert sql 명령어의 작성 방법(구조) :
   insert into 
          테이블이름 ( 컬럼이름1, 컬럼이름2, ... 컬럼이름n)
          values( 값1, 값2, ... 값n ) ;

예) 방금 만든 학생 테이블에 새로운 학생 데이터를 저장하는
    insert sql 명령어를 파일로 저장해 놓기 :
    insert_student_data.sql

3. 테이블에 저장된 데이터를 검색(조회)하기
-> select sql 명령어를 사용하기

4. 테이블에 저장된 데이터를 바꾸기(수정)
-> update sql 명령어를 사용하기

5. 테이블에 저장된 데이터를 없애기(삭제)
-> delete sql 명령어를 사용하기
