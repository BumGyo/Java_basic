* 회원 아이디를 자동으로 만들어 주는 방법 3가지

1. 프로그램에서 정수 값을 만든 후에 테이블에 넣어주기
-> static 제한자를 사용해서 정수형 변수를 선언해야 합니다!!
예) private static int s_member_id = 1; 
// 클래스 별로 만들어지는 변수
// -> 모든 객체들이 함께 사용할 수 있는 변수(공통 또는 공유 변수)

예) private int member_id = 1; // 객체 마다 만들어지는 변수 
// -> 객체 별로 메모리에 따로 생성되는 인스턴스 변수

2. 오라클 데이터베이스의 
   테이블에서 제공해주는 max( ) 함수를 사용하기

3. 오라클 데이터베이스의 시퀀스(Sequence)를 사용하기

* 시퀀스(Sequence) : 규칙을 갖고 있는 숫자를 만들어주는 객체(Object)
